## Stack overview

The 1753 SKINCARE skin analysis is a hybrid system combining three AI layers and seven lifestyle questions. All image processing happens client-side in the user's browser. No image is sent to a remote server unless the user explicitly consents to save it to their encrypted history.

### Layer 1 — Facial landmark detection (client-side)

- Engine: Google MediaPipe Face Landmarker
- Precision: 478 facial landmarks
- Purpose: Precise cropping of 12 zones:
  1. Forehead
  2. Left cheek
  3. Right cheek
  4. Nose
  5. Chin
  6. Left eye area
  7. Right eye area
  8. T-zone
  9. Jawline left
  10. Jawline right
  11. Upper lip area
  12. Neck (when visible)
- Runs on: GPU or CPU delegate via WebAssembly, single-face mode, image mode.

### Layer 2 — Dermatological classification (client-side ONNX)

- Model: MobileNetV3 multi-task classifier
- Training set: 88 000+ dermatological images
- Classifies 9 skin conditions:
  1. Acne (vulgaris)
  2. Dermatitis (contact / seborrheic)
  3. Dryness / xerosis
  4. Eczema (atopic)
  5. Hyperpigmentation
  6. Normal / no condition
  7. Psoriasis
  8. Rosacea
  9. Sun damage (photoaging)
- Severity levels per condition: None / Mild / Moderate / Severe (4 levels)
- Format: ONNX quantised INT8 (~17 MB), runs in browser via ONNX Runtime Web
- Input size: 224×224 RGB per zone
- Model file: /models/skin_classifier_q8.onnx

### Layer 3 — Holistic visual assessment (server-side GPT-5.4 Vision)

- Model: OpenAI GPT-5.4 with vision input
- Role: Independent visual verification, cross-validation of ONNX output, identification of subtle features (texture quality, glow, puffiness, under-eye circles), and synthesis with lifestyle answers.
- System prompt: Holistic, sceptical of mainstream cosmetic claims, recommends lifestyle interventions first (sleep, diet, stress management, movement, gut health) and at most 2–3 products.
- Output: Structured JSON with 15 metrics, biological skin age estimate, Fitzpatrick skin type, personalised recommendations.

## The 15 skin metrics

Each metric is scored 0–100 where 100 is optimal.

1. Hydration
2. Elasticity / firmness
3. Pore condition
4. Texture (smoothness, bumpiness)
5. Evenness (colour uniformity)
6. Sensitivity / reactivity
7. Sebum level
8. Wrinkles / fine lines
9. Dark circles
10. Redness / inflammation
11. Acne score
12. Pigmentation
13. Sun damage
14. Barrier function
15. Overall skin health

## Derived outputs

- **Biological skin age**: estimated ±5 years from chronological age, based on metrics 2, 4, 8, 13, 14.
- **Fitzpatrick type**: I, II, III, IV, V or VI.
- **Radar chart**: 8-axis visualisation of key metrics.
- **Time series**: stored per user, shows metric development over time with AES-256-GCM encryption.

## Lifestyle questions (7)

These contribute ~30% of the final holistic score:

1. Average sleep hours per night
2. Diet quality and composition
3. Daily stress level
4. Movement / exercise frequency
5. Gut health (bowel regularity, known digestive issues)
6. Self-declared skin type
7. Main skin goal

## Holistic framework

The analysis is built on three scientific pillars that are **not** mainstream in standard skincare analyses:

1. **Skin microbiome** — healthy skin depends on commensal species (Staphylococcus epidermidis, Cutibacterium acnes in balance).
2. **Endocannabinoid system (ECS)** — CB1/CB2 receptors regulate sebum, inflammation and keratinocyte proliferation.
3. **Gut-skin axis** — intestinal health directly influences skin inflammation markers.

Recommendations always address lifestyle first, products second. Users receive at most 2–3 product recommendations to avoid over-prescription.

## Privacy and data handling

- 100% client-side AI inference (MediaPipe + ONNX).
- GPT-5.4 call is server-side proxied through 1753's backend to avoid exposing API keys; image is sent base64-encoded, processed, response returned, nothing logged.
- Optional encrypted image history using AES-256-GCM with a user-specific key.
- No third-party analytics on analysis page beyond anonymous pageviews.
- GDPR-compliant.

## Output formats

- In-app visual report with radar chart, per-zone heatmap, metric scoreboard.
- PDF export with branded layout.
- Shareable link (user-controlled, expires after 7 days).

## Differentiators vs. other consumer skin analyses

- More zones analysed (12) than L'Oréal, Olay, Perfect Corp, Revieve.
- More metrics (15) than any consumer competitor.
- Only mainstream consumer analysis with endocannabinoid and gut-skin axis integration.
- Fully client-side image processing vs. cloud-based for competitors.
- No hardware dependency (unlike Neutrogena Skin360).

## Known limitations (honest disclosure)

- No peer-reviewed external clinical validation (yet).
- Requires good lighting (even, soft, front-facing).
- Biological age estimation has ±5 year confidence band.
- Model trained on diverse but not exhaustive skin tones; Fitzpatrick V–VI have slightly lower precision.
- Not a diagnostic tool; recommends dermatologist consultation for any suspected medical condition.
